# Use Cases

## Mode A — Primary Use Cases (tab-first / cover)

### 1. Cover Preparation from Tab or Structured Source

Пользователь имеет Guitar Pro related source, Songsterr-export, `MIDI` или другой compatible structured tab и хочет быстро получить **editable MIDI-oriented REAPER scaffold** вместо ручного построения проекта с нуля.

### 2. Part Reuse from Existing Material

Пользователь переносит отдельные партии (drums/bass/guitar и др.) из существующих structured sources в новый REAPER project scaffold с traceability и маркерами для review.

### 3. Draft Creation from Manual Text/Tab

Пользователь вручную подготавливает text/tab-like input и получает начальный структурированный draft для дальнейшей доработки.

## Mode B — Creative Assistance (structured scaffold only)

### 4. Style-Guided or Idea-First Sketch

Пользователь просит помощь в виде **наброска структуры** («в духе X», «намёк на Y», быстрый каркас ударных/баса): система выдаёт **редактируемые стартовые точки** и arrangement hints, не финальную композицию и не «клон» произведения.

**Non-goals для Mode B:** гарантированная оригинальность на уровне юридической чистоты без review; замена автора; обход лицензий на исходные табы.

## Secondary Use Case

### 5. Screenshot-Assisted Rough Draft

Если structured file недоступен, пользователь подаёт screenshot/tab image и получает warning-heavy rough draft для последующей ручной проверки.

## Non-Use Cases

- автоматическая финализация всей аранжировки «под ключ»;
- замена музыкального решения и ответственности пользователя;
- trustworthy-by-default extraction из raw audio;
- vocal/lyric workflow в MVP;
- инструмент для неэтичного копирования чужих произведений под видом Mode B.
