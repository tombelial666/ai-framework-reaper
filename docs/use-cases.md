# Use Cases

## Primary Use Cases

### 1. Cover Preparation from Structured Source

Пользователь имеет Guitar Pro related source, `MIDI` или `MusicXML` и хочет быстро получить editable REAPER scaffold вместо ручного построения проекта.

### 2. Part Reuse from Existing Material

Пользователь переносит отдельные партии из существующих track sources в новый REAPER project scaffold.

### 3. Draft Creation from Manual Text/Tab

Пользователь вручную подготавливает text/tab-like input и получает начальный структурированный draft для дальнейшей доработки.

## Secondary Use Case

### 4. Screenshot-Assisted Rough Draft

Если structured file недоступен, пользователь подаёт screenshot/tab image и получает warning-heavy rough draft для последующей ручной проверки.

## Non-Use Cases

- автоматическая финализация всей аранжировки;
- замена музыкального решения пользователя;
- trustworthy-by-default extraction из raw audio.
