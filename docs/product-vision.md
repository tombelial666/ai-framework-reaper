# Product Vision

## Vision

Создать framework repository для детерминированной AI-assisted подготовки **editable REAPER project scaffolds** из structured и semi-structured музыкальных источников.

## Product Outcome

Пользователь должен получать не "финальную аранжировку", а:

- понятный черновик проекта;
- подготовленные треки и структуру;
- переносимые данные для REAPER;
- warnings и unresolved markers;
- быстрый вход в ручную доработку.

## User Value

- меньше ручного ввода с нуля;
- быстрее старт для cover preparation и arrangement drafting;
- прозрачные ограничения системы;
- сохранение редактируемости результата.

## Product Positioning

Это не generative music system и не raw-audio transcription engine.

Это deterministic preparation workflow для случаев, когда у пользователя уже есть:

- Guitar Pro related structured source;
- MIDI;
- MusicXML;
- вручную подготовленный text/tab-like source;
- или, во вторичном path, screenshot/tab image.

## Confirmed

- Target DAW: `REAPER`.
- **Human-in-the-loop**: manual review is mandatory.
- Structured-core path is primary.

## Deferred

- Полноценная интеллектуальная orchestration или composition assistance.
- Audio-first workflows.
