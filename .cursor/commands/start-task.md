# Start Task

Use this command when beginning substantial work in this repository.

## Steps

1. Read `MANIFEST.md`.
2. Read the relevant docs in `docs/`.
3. Create or update `tasks/current-task.md` from `tasks/_template.md`.
4. Classify the task into `Confirmed`, `Assumption`, `Risk`, `Deferred`, `Open Question`.
5. If the task changes architecture or scope, update docs before implementation.
