# Ingest Score

Use this command when analyzing or planning a score ingestion task.

## Workflow

1. Identify source type: Guitar Pro related source, `MIDI`, `MusicXML`, manual text/tab-like input, or screenshot.
2. Validate whether the source belongs to MVP scope.
3. Decide whether the path is structured-core or screenshot-secondary.
4. Define parsed output, metadata capture and warning strategy.
5. Define how parsed data will map into the canonical model from `docs/internal-data-model.md`.
6. Record assumptions, risks and open questions instead of guessing missing capabilities.
